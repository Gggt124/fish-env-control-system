# คู่มือการติดตั้ง Firmware สำหรับ Fish Pump

โฟลเดอร์นี้บรรจุ Firmware และโปรแกรมสำหรับอัปเดตบอร์ดควบคุมปั๊มน้ำของคุณ

## สิ่งที่ต้องเตรียม
1. คอมพิวเตอร์ระบบปฏิบัติการ **Windows 10 หรือ Windows 11**
2. สาย USB สำหรับเสียบเชื่อมต่อกับบอร์ด ESP32

## วิธีการอัปเดต (Flash)
1. เสียบสาย USB ระหว่างคอมพิวเตอร์และบอร์ด ESP32
2. ดับเบิลคลิกไฟล์ **`FLASH.bat`**
3. โปรแกรมจะรันหน้าต่างสีดำ (Command Prompt / PowerShell)
   - เลือกรุ่นบอร์ดของคุณ: **[1]** ESP32 Classic หรือ **[2]** ESP32-S3
   - หากโปรแกรมถามหา **COM port** ให้พิมพ์เลขพอร์ตที่ปรากฏ (เช่น `COM5`) แล้วกด Enter
   - ไม่จำเป็นต้องติดตั้งโปรแกรมเพิ่มเติม — เครื่องมือทั้งหมดรวมอยู่ในโฟลเดอร์นี้แล้ว
4. รอจนกว่าจะขึ้นคำว่า **"[OK] Flash Successful!"**
5. โปรแกรมจะแสดง **Wi-Fi Password** สำหรับบอร์ดของคุณ ให้จดหรือถ่ายรูปเก็บไว้!

## หากลืมรหัสผ่าน Wi-Fi (AP Password)
รหัสผ่าน Wi-Fi จะถูกสุ่มขึ้นมาเฉพาะสำหรับบอร์ดของคุณ (อ้างอิงจาก MAC Address)
หากคุณลืมรหัสผ่าน:
1. เสียบสาย USB เข้ากับบอร์ด
2. ดับเบิลคลิกไฟล์ **`SHOW_PASSWORD.bat`**
3. โปรแกรมจะอ่านข้อมูลจากบอร์ดและแสดงรหัสผ่านให้คุณทราบโดยไม่ต้องอัปเดต Firmware ใหม่
