=======================================================
   คู่มือการติดตั้ง FIRMWARE สำหรับ FISH PUMP
=======================================================

โฟลเดอร์นี้บรรจุ Firmware และโปรแกรมสำหรับอัปเดตบอร์ด
ควบคุมปั๊มน้ำของคุณ

-------------------------------------------------------
 ข้อควรระวังสำหรับบอร์ด ESP32-S3
-------------------------------------------------------
ต้องเสียบสาย USB เข้าที่ช่อง "COM" (หรือ "UART") เท่านั้น
ห้ามเสียบช่อง "USB"

พอร์ต COM/UART จะรีเซ็ตบอร์ดเข้าโหมดอัปเดตโดยอัตโนมัติ
ไม่ต้องกดปุ่มใดๆ เพิ่มเติมครับ

-------------------------------------------------------
 สิ่งที่ต้องเตรียม
-------------------------------------------------------
1. คอมพิวเตอร์ระบบปฏิบัติการ Windows 10 หรือ Windows 11
2. สาย USB สำหรับเสียบเชื่อมต่อกับบอร์ด ESP32

-------------------------------------------------------
 วิธีการอัปเดต (FLASH)
-------------------------------------------------------
1. เสียบสาย USB เข้าที่ช่อง "COM" (หรือ "UART") บนบอร์ด ESP32-S3
   (สำหรับ ESP32 Classic เสียบพอร์ตเดียวที่มีได้เลย)
2. ดับเบิลคลิกไฟล์ FLASH.bat
3. โปรแกรมจะรันหน้าต่างสีดำ (Command Prompt / PowerShell)
   - เลือกรุ่นบอร์ดของคุณ: [1] ESP32 Classic หรือ [2] ESP32-S3
   - หากโปรแกรมถามหา COM port ให้พิมพ์เฉพาะตัวเลขพอร์ต
     ที่ปรากฏ (เช่น 5) แล้วกด Enter
4. รอจนกว่าจะขึ้นคำว่า "[OK] Flash Successful!"
5. โปรแกรมจะแสดง Wi-Fi Password สำหรับบอร์ดของคุณ 
   ให้จดหรือถ่ายรูปเก็บไว้!

-------------------------------------------------------
 หากลืมรหัสผ่าน WI-FI (AP PASSWORD)
-------------------------------------------------------
รหัสผ่าน Wi-Fi จะถูกสุ่มขึ้นมาเฉพาะสำหรับบอร์ดของคุณ (อ้างอิงจาก
MAC Address) หากคุณลืมรหัสผ่าน:
1. เสียบสาย USB เข้ากับบอร์ด
2. ดับเบิลคลิกไฟล์ SHOW_PASSWORD.bat
3. โปรแกรมจะอ่านข้อมูลจากบอร์ดและแสดงรหัสผ่านให้คุณทราบ
   โดยไม่ต้องอัปเดต Firmware ใหม่
