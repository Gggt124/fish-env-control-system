<objective>
Change firmware boot so pump control uses persisted settings and obeys persisted auto-start.

Purpose: Make reboot behavior reflect saved timer settings, relay polarity, and auto-start while keeping the device reachable if settings are invalid or pump startup fails.
Output: `app_main.c` initializes NVS before pump control, resolves persisted pump settings into `pump_control_config_t`, initializes pump control stopped, and starts it only when auto-start is enabled and saved settings are valid or missing/defaulted.
</objective>
