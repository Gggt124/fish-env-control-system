## Critical Anti-Patterns (do NOT repeat these)

- [MAX_PATH Build Error]: Running `idf.py build` in deeply nested folders breaks dependency generation. → [Mitigation]: Run build in a custom shorter path, or ensure our local build script (`build.ps1`) works correctly within Windows limits.
