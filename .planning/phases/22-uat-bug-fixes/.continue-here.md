## Critical Anti-Patterns (do NOT repeat these)
- [Encoding Failure]: Previous agents used default PowerShell string encoding to overwrite files, breaking UTF-8 Thai strings. → [Mitigation]: Use Node.js `fs` module with `'utf8'` or standard native agent tools (`write_to_file` / `replace_file_content`) to write files.
- [MAX_PATH Build Error]: Running `idf.py build` in deeply nested folders breaks dependency generation. → [Mitigation]: Run build in a custom shorter path, e.g., `idf.py -B C:\Users\Copter\.gemini\antigravity\b build`.
