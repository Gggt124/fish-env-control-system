<task type="auto">
  <name>Task 2: Web Server Cookie Management</name>
  <files>main/web_server.c</files>
  <action>
    In `handle_api_login`, parse the `"remember"` boolean from the incoming cJSON payload (defaulting to false).
    After calling `session_create` successfully, construct the `Set-Cookie` header.
    If `remember` is true (AUTH-01), append `; Max-Age=31536000` (1 year) to make it a persistent cookie. If false, omit `Max-Age` so the browser treats it as a volatile session cookie.
    In `handle_api_logout`, update the `Set-Cookie` header to use `Max-Age=0` to instruct the browser to delete the cookie immediately.
  </action>
  <verify>
    <automated>idf.py build</automated>
  </verify>
  <done>Web server respects the remember flag and issues correct cookie headers.</done>
</task>
