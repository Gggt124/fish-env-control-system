  <files>components/nvs_store/nvs_store.c, main/app_main.c</files>
  <action>
    Modify `nvs_store_init()` to implement SEC-01. When calling `nvs_flash_init()`, check the return value. If it returns `ESP_ERR_NVS_NO_FREE_PAGES` or `ESP_ERR_NVS_NEW_VERSION_FOUND`, call `nvs_flash_erase()` and then retry `nvs_flash_init()`. Check `main/app_main.c` to see if it calls `nvs_flash_init` directly; if it does, replace it with `nvs_store_init()` and make sure the error handling happens only in `nvs_store_init()`.
