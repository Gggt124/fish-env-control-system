3. In `update_control_locked()`:
   - If the mode is `COOLING_CONTROL_MODE_FORCE_OFF`, set `s_auto_demand = false;`.

   - If the sensor is not ready (`!cooling_sensor_ready_locked()`), set `s_auto_demand = false;`.
   - In the `COOLING_CONTROL_MODE_AUTO` branch:
     - Check `s_auto_demand` instead of `s_relay_energized` to decide whether to evaluate hysteresis:
       - If `s_auto_demand` is true, check `demand = s_temperature_c >= off_c;`.
       - If `s_auto_demand` is false, check `demand = s_temperature_c >= threshold_c;`.
     - Update `s_auto_demand = demand;` after evaluating.
