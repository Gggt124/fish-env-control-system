<threat_model>
### T-14-05: Remote dependency breaks SoftAP operation
- Severity: medium
- Surface: embedded frontend
- Risk: CDN scripts, fonts, icons, or remote images fail when the ESP32 AP has no internet.
- Mitigation: search static assets for remote dependencies and document any intentional URL.
- Verification: `footprint.md` includes the exact search command and results.

### T-14-06: UI asset growth threatens OTA partition
- Severity: medium
- Surface: firmware image and partition layout
- Risk: embedded static asset growth can push the firmware image toward the `0x1F0000` app slot limit.
- Mitigation: build firmware and calculate binary size against the app slot.
- Verification: `footprint.md` records binary bytes, slot bytes, and free headroom.
</threat_model>
