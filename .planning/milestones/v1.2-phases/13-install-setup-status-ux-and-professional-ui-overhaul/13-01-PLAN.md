# Phase 13, Plan 1: Complete CSS Overhaul (Premium Dark Glassmorphism & Bento Grid)

## Goal
Completely overhaul `style.css` to deliver a professional Premium Dark/Glassmorphism theme with a Bento Grid layout, conforming to the decisions in `13-CONTEXT.md` (D-01, D-02, D-03, D-07, D-08).

## Steps

1.  **Define Dark Glassmorphism HSL Variables:**
    - Open `main/static/style.css`.
    - Update CSS variables (`:root`) to reflect a sleek dark indigo/slate theme.
    - Set `--bg-body` to `#0a0b10` or similar deep dark indigo.
    - Set `--surface-container-lowest` to a semi-transparent glass color (e.g. `rgba(255, 255, 255, 0.05)` or `rgba(15, 17, 26, 0.6)`).
    - Set borders (`--outline-variant`, `--surface-container-highest`) to subtle translucent values (e.g. `rgba(255, 255, 255, 0.1)`).
    - Update primary text (`--on-surface`) to bright white or near-white, and secondary text (`--on-surface-variant`) to light gray/slate.
    - Set primary accent (`--primary`) to neon/indigo (e.g., `#5856d6`).
    - Keep semantic status colors: `--secondary` (green), `--tertiary` (orange), `--error` (red).

2.  **Implement Glassmorphism Properties:**
    - Add `backdrop-filter: blur(12px)` to cards, panels, and sidebars (like `.card`, `.app-sidebar`, `.section-card`, `.mobile-topbar`).
    - Ensure borders are thin and translucent to give the glass effect.
    - Add subtle inner shadows or highlights if possible without external assets.

3.  **Implement Bento Grid Layouts:**
    - Refine `.status-grid` and `.summary-grid` to ensure cards are presented in a clean, masonry-like Bento Grid. Use CSS Grid with gap.
    - Adjust padding and margins to feel spacious yet contained.

4.  **Enhance Accessibility (Focus & Touch Targets):**
    - Ensure `min-height: 44px` and `min-width: 44px` for interactive elements (buttons, links, inputs).
    - Update `:focus-visible` to an unbroken 3px solid `--primary` ring with `outline-offset: 2px`.

5.  **Implement Reduced Motion:**
    - Add a `@media (prefers-reduced-motion: reduce)` block at the end of `style.css`.
    - Disable or shorten transition durations, pulse animations, and slide-in animations.

## Verification
- Verify that the app shell loads with the dark theme and glassmorphism styling.
- Verify focus rings are clearly visible via keyboard navigation.
- Ensure no external fonts or images were introduced.
