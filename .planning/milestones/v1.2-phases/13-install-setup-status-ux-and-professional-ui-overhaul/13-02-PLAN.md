# Phase 13, Plan 2: Wi-Fi Setup UX (Keyboard Semantics & Reconnection Banner)

## Goal
Improve the Wi-Fi setup experience by implementing keyboard-operable semantic buttons for network rows (D-04) and a persistent reconnection helper banner (D-06) during APSTA transitions, without changing backend functionality.

## Steps

1.  **Semantic Button Rows for Wi-Fi Networks:**
    - Open `main/static/app.js`.
    - Locate `renderWifiList()`.
    - Change the generated `<li>` tags for each network to `<button class="network-item" type="button">`.
    - Ensure `onclick="selectNetwork('...')" ` works.
    - Add focusability so the elements receive the `:focus-visible` ring.
    - Open `main/static/style.css` and update the styles for `.network-item` (previously an `li` inside `.network-list`) to remove default button styles (background, border) and apply the glassmorphism list styles, keeping text alignment correct.

2.  **Reconnection Helper Banner (Offline APSTA Handling):**
    - Open `main/static/wifi.html`.
    - Add a new hidden banner element `#reconnect-banner` inside the main content or `.input-panel`.
    - The banner should have `role="alert"` (or `status`) and display Thai text: "กำลังเชื่อมต่อ... หากหลุด ให้ตรวจสอบ Wi-Fi มือถือว่าต่อกับ SoftAP IP 192.168.4.1 อีกครั้ง" (Connecting... If disconnected, please check if your phone is connected to SoftAP IP 192.168.4.1). Include a countdown timer.
    - Open `main/static/app.js`.
    - In `doConnect()`, show the `#reconnect-banner` and start the countdown timer (e.g., 15 seconds) to set user expectations during the connection attempt.
    - In `pollWifiConnection()` and `onConnectSuccess()`/`onConnectFail()`, hide the banner when a definitive response is received.

3.  **Refine Setup Feedback States:**
    - Ensure scanning shows a polite loading status using `aria-live="polite"`.
    - Ensure errors display in a `role="alert"` container (e.g. `connect-status`).

## Verification
- Press `Tab` on the Wi-Fi page and verify that networks can be focused and selected with `Space`/`Enter`.
- Click connect and observe the reconnection helper banner with the 192.168.4.1 fallback instruction.
