### 2. ข้อมูลที่ต้องกรอก (Form Fields)
**Options Presented:**
- ขอแค่ รหัสผ่านเดิม (Current Password) ก็พอ (ผู้ใช้น่าจะจำ Username ปัจจุบันได้อยู่แล้ว เน้นเปลี่ยนง่ายๆ)
- ขอทั้ง Username เดิม และ Password เดิม เพื่อความชัวร์ก่อนให้เปลี่ยนรหัส
- ต้องตั้ง Username ใหม่ และ Password ใหม่ ไปพร้อมกันเลยในรอบเดียว

**User Selected:** ขอแค่ รหัสผ่านเดิม (Current Password) ก็พอ (ผู้ใช้น่าจะจำ Username ปัจจุบันได้อยู่แล้ว เน้นเปลี่ยนง่ายๆ)

**Notes:** Server สามารถดูด Username จาก Session ID เพื่อเปรียบเทียบได้เลย ไม่ต้องให้ผู้ใช้พิมพ์ซ้ำ
