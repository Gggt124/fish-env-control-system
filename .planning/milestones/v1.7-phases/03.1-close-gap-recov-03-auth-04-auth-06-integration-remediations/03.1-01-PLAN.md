<objective>
This plan implements integration remediations to close gaps for AUTH-04, AUTH-06, and RECOV-03.
Specifically, it removes IP binding from session validation (per D-01), implements global session invalidation (per D-02) and integrates it when credentials change are confirmed (per D-03), ensures staged credentials changes are properly committed in NVS (per D-04), and creates the missing Phase 2 verification report (per D-06). Wi-Fi connection and profile saves bypass staging per D-05.
</objective>
