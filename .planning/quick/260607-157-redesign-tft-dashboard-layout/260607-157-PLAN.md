# Plan: Redesign TFT Dashboard Layout

## Must-Haves
- Status bar includes the mDNS hostname (`fish-pump.local`) alongside the IP.
- Pump status displayed as a large, filled color block (Green/Gray) using a double-size font.
- Countdown includes a visual horizontal progress bar that fills/empties based on `(total_duration - remaining) / total_duration * 100`.
- Left and right column panels use a dark navy background (#0A0A1A / `0x0821`).
- Temperature uses a double-size font with threshold-based color coding (Green < 28, Yellow 28-30, Red > 30).
- Phase indicator uses a solid green block for ON and dark/dim for OFF instead of text.
- Cooling relay status uses a cyan/blue color indicator for ON and gray text for OFF.
